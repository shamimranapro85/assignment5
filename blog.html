Important FAQ

what is the document object model (dom )?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of HTML or XML as a tree of objects, allowing developers to manipulate the content, structure, and styling of a webpage dynamically using JavaScript. Each element, attribute, and text becomes accessible for changes.

How do you select an element from the Dom?

we can select a DOM element using methods like getElementById, querySelector, or getElementsByClassName in JavaScript.

what is event delegation is the context of the dom , and why is it useful?

Event delegation allows you to handle events at a parent element instead of individual child elements. It improves efficiency by reducing the need for multiple event listeners, especially in dynamic content.

how do you manipulate an element's attributes and styles using the dom?

We can manipulate an element's attributes with methods like setAttribute() and removeAttribute(). For styles, use element.style.property to modify inline CSS directly, e.g., element.style.color = "blue";.
